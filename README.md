#### Ejercicio 1

Crear una clase que represente un vehículo, que tenga las siguientes propiedades y métodos:
_Propiedades:_

- Color
- Marca
- Modelo
- Placas
- Encendido (Prendido o Apagado)
- Gasolina

_Métodos:_

- Encender
- Apagar
- Recargar

**Nota:** En el metodo _Encender_ verifica si hay gasolina antes de encender
